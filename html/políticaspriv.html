Funerária La Paz - Política de Privacidade

Na Funerária La Paz, temos um compromisso inabalável com a proteção dos dados pessoais de nossos clientes, colaboradores e parceiros. Esta Política de Privacidade tem como objetivo esclarecer como coletamos, utilizamos, armazenamos e protegemos as informações fornecidas a nós, em conformidade com a Lei Geral de Proteção de Dados (LGPD) - Lei nº 13.709/2018.

1. Dados Coletados

Podemos coletar os seguintes tipos de informações:

1.Dado pessoais: Nome, CPF, RG, endereço, telefone, e-mail, e outros dados necessários para a prestação de nossos serviços.

2. Dados financeiros: Informações de pagamento, como dados bancários ou de cartão de crédito, para a execução de contratos.

3. Dados sensíveis: Informações relacionadas ao estado de saúde, filiação ou outras circunstâncias relacionadas ao falecimento, quando necessário para nossos serviços.

2. Finalidades do Uso dos Dados

Os dados coletados são utilizados exclusivamente para os seguintes fins:

- Prestação de serviços funerários, incluindo organização de cerimônias, sepultamento ou cremação.

- Emissão de documentos legais, como certidões de óbito e contratos.

- Comunicação com clientes e familiares para esclarecimentos ou acompanhamento dos serviços.

- Cumprimento de obrigações legais e regulatórias.

3. Compartilhamento de Dados

A Funerária La Paz pode compartilhar dados pessoais com:

- Autoridades governamentais, para emissão de certidões ou cumprimento de exigências legais.

- Parceiros ou prestadores de serviços contratados para executar parte de nossas atividades, como floriculturas ou transporte funerário, sempre respeitando os limites desta política.

- Instituições financeiras, para processamento de pagamentos.

Em nenhuma circunstância vendemos ou comercializamos os dados pessoais de nossos clientes.

4. Segurança dos Dados

Implementamos medidas técnicas e organizacionais rigorosas para proteger os dados pessoais contra acesso não autorizado, perda, alteração ou destruição, como:

- Armazenamento seguro em servidores protegidos.

- Restrição de acesso a informações sensíveis apenas a colaboradores autorizados.

- Criptografia de dados financeiros e outras informações sensíveis.

5. Direitos dos Titulares dos Dados

Os titulares têm garantidos os seguintes direitos, conforme previsto pela LGPD:

- Confirmar a existência de tratamento de dados.

- Acessar, corrigir ou atualizar informações pessoais.

- Solicitar a exclusão de dados, quando possível e legalmente permitido.

- Revogar o consentimento para uso de dados pessoais.

6. Retenção de Dados

Os dados pessoais serão mantidos apenas pelo período necessário para cumprir as finalidades descritas nesta política, exceto quando houver necessidade de retenção para cumprimento de obrigações legais ou regulatórias.

7. Alterações na Política de Privacidade

A Funerária La Paz reserva-se o direito de atualizar esta Política de Privacidade a qualquer momento, com aviso prévio aos clientes, sempre garantindo a transparência e o alinhamento com a legislação vigente.
